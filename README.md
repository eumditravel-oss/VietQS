# VietQS

Viet QS homepage draft.

This is a static one-page website concept inspired by the section-based business page interaction of LOTTE's corporate site and reworked for Viet QS's construction quantity surveying brand.

## Structure

- `index.html` - homepage markup
- `assets/css/styles.css` - responsive styling
- `assets/js/main.js` - service tab and menu interactions
- `assets/images/` - Viet QS logos and selected construction/estimating visuals

## Content Basis

The service content was drafted from the provided legacy website data:

- quantity take-off
- construction cost review
- drawing change settlement
- construction claims
- overseas / FED estimating
- CAD-based quantity surveying

## Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.
